# Vegalicous Backend API

Backend about Vegalicous API develop using [ExpressJS](https://expressjs.com/) database using PostgreSQL

# How to Install

## Normal Install

    $ npm install
    $ npm migarte
    $ npm start

## Install with docker

    docker-compose up --build
